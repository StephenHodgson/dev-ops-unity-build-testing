Class AudioInfluencerController

Class which supports components implementing being used with audio sources.

Inheritance
Object
AudioInfluencerController
Namespace: Microsoft.MixedReality.Toolkit.SDK.Audio.Influencers
Assembly: cs.temp.dll.dll
Syntax
public class AudioInfluencerController : MonoBehaviour
Remarks

AudioInfluencerController requires an component. If one is not attached, it will be added automatically. Each sound playing game object needs to have an AudioInfluencerController attached in order to have it's audio influenced.
